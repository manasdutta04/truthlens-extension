TruthLens API

Welcome to TruthLens API

TruthLens is an AI-powered fake news detection system. This API provides endpoints for analyzing news articles, detecting bias, sentiment, and verifying credibility through trusted sources.

Article Analysis

Analyze an article for credibility:

POST /api/analyze

Using the API

The API is designed to be used with the TruthLens browser extension, but can also be used directly. Check the API documentation for detailed information on each endpoint.

Sample Request


curl -X 'POST' \
  'https://truthlens-ii4r.onrender.com/api/analyze' \
  -H 'Content-Type: application/json' \
  -d '{
  "title": "Sample Article Title",
  "content": "This is the content of the article to analyze.",
  "url": "https://example.com/sample-article"
}'
